# ニュースレポートアプリ

## 概要
Google NewsのRSSフィードから最新ニュースを自動収集し、カテゴリ別に表示するWebアプリケーションです。ユーザーは興味のあるキーワードを登録して、パーソナライズされたニュースを閲覧できます。

## 主な機能
- カテゴリ別ニュース一覧表示（ビジネス、テクノロジー、科学、健康、スポーツ、エンターテイメント）
- マイキーワード登録機能
- キーワードに基づいたパーソナライズニュース表示
- 記事のソース元・公開日時表示
- キーワードごとの記事フィルタリング
