# ニュースレポートアプリ

## 概要
Google NewsのRSSフィードから最新ニュースを自動収集し、日本語で表示するWebアプリケーションです。

## 主な機能
- カテゴリ別ニュース一覧表示
- 記事詳細表示（要約・全文表示）
- 非同期での記事本文取得

## 使用技術
### バックエンド
- Python 3.10+
- Django 4.2
- Celery（非同期タスク処理）
- Redis（メッセージブローカー）

### フロントエンド
- HTML5 / CSS3
- Bootstrap 5
- Font Awesome

### データベース
- SQLite（開発環境）
